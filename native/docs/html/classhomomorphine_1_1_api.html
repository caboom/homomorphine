Detailed Description

/brief Provides a RESTFul API interface.

This class provides a RESTFul API interface for Homomorphine, it is very simple and does not expose a full functionality of Homomorphine, but rather selected group of functionalities. It is not advised to use this interface except for testing Homomorphine functionalities (RESTFul interface will be provided by the Golang wrapper around Homomorphine).
